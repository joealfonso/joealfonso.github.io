Calendar Application Prototype


An application prototype I did on my own time to demonstrate and practice certain mobile patterns and specifically addressing the Android design environment. The main objective of the calendar view was to provide the user with a sense of their agenda, not necessarily identify which day it was. This is demonstrated by the choice to highlight the days with events more prominently than the current day which is treated with a change in the number color only, while leaving the cell white. The number of events are displayed by little notches on the left side of the highlighted days providing the user with information about the number of the events as well without having to switch to a day view or tapping the day itself for additional details.
